# RAG 智能助手自动化测试工具 v3.0

这是一个基于 Python 和 DeepSeek LLM 的自动化测试平台，旨在评估 RAG (检索增强生成) 系统的性能。

## 功能特性

- **多格式知识库支持**：支持导入 .txt, .md, .pdf, .docx, .xlsx 等多种格式的文档作为知识库。
- **智能测试用例生成**：利用 LLM 阅读知识库，自动生成包含参考答案和评分标准的测试用例。
- **全流程解耦**：
    - **Step 1 生成**：自定义难度、数量、侧重点。
    - **Step 2 获取回答**：支持内置模拟器（含对抗模式：幻觉/冗长）或导入外部真实系统回答。
    - **Step 3 智能评分**：LLM 裁判基于多维度（忠实度、完整性、相关性）进行打分。
- **可视化报告**：生成 HTML 雷达图和详细评分理由。
- **Debug 模式**：提供详细的 API 交互日志，方便排查问题。

## 安装指南

1. 确保 Python 3.8+ 环境。
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

## 使用说明

1. **启动工具**：
   ```bash
   python run_tool.py
   ```
2. **配置知识库**：
   - 在界面选择 `knowledge_base` 文件夹（或将文档放入其中）。
3. **执行测试**：
   - 点击 **Step 1 生成**，配置参数后生成测试集。
   - 点击 **Step 2 模拟**，选择模拟风格（或导入外部 JSON）。
   - 点击 **Step 3 评分**，等待评估完成。
4. **查看报告**：点击“报告”按钮查看可视化结果。

## 目录结构

- `knowledge_base/`: 存放测试文档。
- `outputs/`: 存放生成的所有数据（测试集、回答、报告）。
- `src/`: 源代码。
